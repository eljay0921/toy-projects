# Java와 Kotlin 비교 학습 프로젝트

이 프로젝트는 Java와 Kotlin의 문법과 특징을 비교하며 학습하기 위한 저장소입니다.

## 학습 목표

- Java와 Kotlin의 기본 문법 비교
- 두 언어의 특징과 차이점 이해
- 실제 코드 예제를 통한 실습

## 주요 학습 내용

- 기본 문법 및 변수 선언 : 확장함수
- 조건문과 반복문
- Null Safety
- 예외 처리
- 함수형 프로그래밍 : 람다와 고차함수
- 비동기 & 병렬 처리 : 코루틴
- 제네릭

## 프로젝트 구조

```
java-kotlin-project/
├── src/
│   ├── main/java/example/java/     # Java 예제 코드
│   └── main/java/example/kotlin/   # Kotlin 예제 코드
```
